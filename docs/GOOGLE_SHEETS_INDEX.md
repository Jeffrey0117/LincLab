## 新欄位順序速查

**重要！新的欄位順序如下：**

```
A 欄 (第 1 個) → 時間 (created_at)
                範例：2024-11-19 10:30:00

B 欄 (第 2 個) → 標題 (title)
                範例：我的商品

C 欄 (第 3 個) → 蟑螂分潤網連結 (short_url)
                格式：https://linclab.vercel.app/{short_code}
                範例：https://linclab.vercel.app/abc123

D 欄 (第 4 個) → 圖片URL (og_image)
                範例：https://example.com/image.jpg
```
